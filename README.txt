Compilation/Run Instructions: An interactive desktop simulator designed to model and execute the software measurement process based on the ISO/IEC 15939 standard. Built using Java Swing, the application features a 5-step wizard interface (Profile, Define, Plan, Collect, Analyse) that automates metric calculation, performs gap analysis, and visualizes multi-dimensional software quality data using a custom Java 2D radar chart.
This project runs entirely on standard Java SE with no external library dependencies.
To compile the project from the command line, navigate to the project directory and run: javac src/*.java -d out
To run the project, run: java -cp out Main

You can find the step-by-step screenshots (step1_profile.png to step5_analyse.png) showing the fully functional wizard steps and the custom radar chart directly in the root directory of this repository.
